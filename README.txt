AlphaZero for Kalaha and TicTacToe
=========

This directory contains pytorch implementation of AlphaZero as well as two environments to test it on.

## Command:

python3 main.py <env>
    <env> is optional parameter. It is set to "kalaha" by default 
    <env> = {"kalaha", "tictactoe"}

All AlphaZero implementation is contained in "alphazero" directory.
